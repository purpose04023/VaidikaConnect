# **App Name**: VaidikaConnect

## Core Features:

- Puja Program Listing: Display a searchable list of Vaidika programs (e.g., Satyanarayana Vratham, Ganapathi Homam) with a search bar.
- Participant Input & Pujari Filtering: Allow users to input the number of participants for a selected puja, and use this input to filter eligible Pujaris.
- Map-Based Pujari Discovery: Display a Google Maps view centered on the user's location, showing markers for Pujaris who can perform the selected puja and accept the specified number of members. Markers link to Pujari profiles.
- Bottom Scroll Panel: Implement a scrollable bottom sheet synced with the map, showing Pujari photo, name, rating, price, and a chat button.
- Pujari Profile: Display full details including profile photos, Vedic qualifications, languages, experience, list of pujas, pricing, availability, reviews, phone number, and in-app chat option.
- In-App Communication: Enable in-app chat between user and Pujari. Allow direct phone calls (with user permission).
- Pujari Recommendation: Suggest to the user the Pujaris who are likely to be the best match based on user preferences and the profile of other similar users using a recommendation tool.

## Style Guidelines:

- Primary color: Saffron (#FF9933) to reflect the sacredness and cultural significance of Hindu traditions.
- Background color: Light beige (#F5F5DC), a desaturated version of saffron, providing a soft, neutral backdrop.
- Accent color: Warm gold (#D4AF37), analogous to saffron, to highlight important elements such as ratings and prices.
- Body font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality, suitable for body text.
- Headline font: 'Alegreya', a humanist serif with an elegant, intellectual, contemporary feel; suitable for headlines
- Use traditional Hindu symbols and motifs for icons.
- Clean and spacious layout to ensure easy navigation and readability.
