# Context Tracker

## Base Project Structure
The project is a Next.js TypeScript application.
- `src`: Application source code (React components, routing, services)
- `supabase`: Database migrations, schemas, configurations
- `docs`: Documentation files
- `scripts`: Helper scripts
- `public`: Static assets

## Discovered Files and Directories
- `package.json`: Main dependency file
- `tsconfig.json`: TypeScript configuration
- `next.config.ts`: Next.js configuration
- `tailwind.config.ts`: Tailwind CSS configuration
